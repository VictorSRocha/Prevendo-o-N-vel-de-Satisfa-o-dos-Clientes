1. Descrição do Projeto¶

Para este projeto será utilizada a linguagem Python e o dataset disponível pela competição Santander Customer Satisfaction, através do Kaggle.

A satisfação do cliente é uma medida fundamental de sucesso. Clientes insatisfeitos cancelam seus serviços e raramente expressam sua insatisfação antes de sair. Clientes satisfeitos, por outro lado, se tornam defensores da marca!

O Banco Santander necessita identificar clientes insatisfeitos no início do relacionamento. Isso permitirá que o Santander adote medidas preventivas para melhorar a felicidade de um cliente antes que seja tarde demais.

Nesse projeto trabalharemos com centenas de recursos anônimos para prever se um cliente está satisfeito ou insatisfeito com sua experiência bancária.

1.1 Objetivo¶

Ao projeto será necessário a criação de um modelo de machine learning que preveja o nível de satisfação dos clientes do Santander ou seja , se o cliente está satisfeito.

Será considerado um algoritmo válido caso tenha uma Acurácia de 70%.

def correlation(threshold):
    global correlacoes
    # Criando uma variável do tipo set
    colunas_rela = set()
    for i in range(len(correlacoes.columns)):
        for j in range(i):
            if abs(correlacoes.iloc[i, j]) > threshold: # necessitamos dos valores absolutos
                colname = correlacoes.columns[i]  # salvando o nome da coluna
                colunas_rela.add(colname)
    return colunas_rela

No pré-processamento precisamos realizar algumas transformações nas variáveis, pois muitos algoritmos de ML exigem que os dados esteja, Padronizados, Normalizados.

Normalização¶

A normalização é quando colocamos os dados em uma escala com intervalo entre 0 e 1.

Padronização¶

Padronização é a técnica para transformar os atributos com distribuição Gaussiana (normal) e diferentes médias e desvios padrão em uma distribuição Gaussiana com a média igual a 0 e desvio padrão igual a 1. Funciona bem quando os dados já estão na mesma escala (normalização).
